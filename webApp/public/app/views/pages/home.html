Welcome to our Evolution web application!

If you're looking to learn something about software evolution, you've come to the right place. This is the place where you can either learn some interesting laws and facts about evolution via the Education Center, or execute some codebase analysis tools from the Research Center!

Don't know where to start? If you're looking for an easy read, check out the blog posts, papers, or books on Software Engineering in the Education center. If you're looking for some cool questions (with answers) to the evolution of various open-source projects, check out the ones included in the Research Center! You'll be able to execute an analysis tool on our server and display the results in your browser. Finally, if you just want more information about this project, check out the 'About' page or contact us.
